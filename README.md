# JEE Advanced 2026 — Personal Study Tracker PWA

A personal Progressive Web App (PWA) for tracking JEE Advanced 2026 preparation.

## 🚀 Setup on GitHub Pages

### Step 1 — Create Repository
1. Go to [github.com](https://github.com) → **New repository**
2. Name it: `jee-tracker` (or anything you like)
3. Set to **Public** (required for free GitHub Pages)
4. Click **Create repository**

### Step 2 — Upload Files
Upload all these files to the repo root:
```
index.html
manifest.json
sw.js
icons/
  icon-192.svg
  icon-512.svg
```

### Step 3 — Enable GitHub Pages
1. Go to repo **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)**
4. Click **Save**

### Step 4 — Access Your App
Your app will be live at:
```
https://YOUR_USERNAME.github.io/jee-tracker/
```
*(Takes ~2 minutes to deploy)*

### Step 5 — Install as App (Optional)
- **Android**: Open in Chrome → Menu → "Add to Home Screen"
- **iOS**: Open in Safari → Share → "Add to Home Screen"
- **Desktop**: Open in Chrome → Address bar install icon
